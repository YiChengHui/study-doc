## 需求弹性
>   指一种商品的<red>需求量对其影响因素变动的反应敏感程度</red>。

影响需求的因素：
商品的价格：需求价格弹性  
消费者收入水平：需求收入弹性  
相关商品的价格：需求交叉弹性  
消费者偏好

### 需求价格弹性
>   一种商品需求量变动对其价格变动的反应程度。一般用需求价格弹性<red>系数</red>来表示需求价格弹性的<red>大小</red>。
